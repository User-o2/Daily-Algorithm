# Daily Algorithm

[![Language](https://img.shields.io/badge/Language-C++20-00599C.svg)](https://isocpp.org/)

## 📌 建立初衷

作为人工智能专业的学生，在进行 LLMs 等方向的探索之余，通过每日的算法训练，保持对数据结构与算法的熟练度。

本项目旨在为各类算法竞赛与保研机试夯实基础，记录每一步代码积累。

## 🗂️ 目录结构

仓库按照模块化与平台化进行分类，确保代码可追溯与高复用性：

```text
.
├── Template/          # 核心算法模板（如：KMP、线段树、图论最短路等）
├── Problems/          # 分平台题解与实战代码
│   ├── CCF-CSP/       # CSP 认证真题还原
│   ├── LeetCode/      # 核心常考题型分类
│   └── Others/        # 洛谷、PAT 等其他 OJ 平台
├── Notes/             # 算法推导笔记与 C++ 现代特性总结
└── .vscode/           # 本地编译与调试配置文件 (Tasks & Launch)

```
